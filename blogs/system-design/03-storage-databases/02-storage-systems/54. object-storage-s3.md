# Object Storage with S3

## Overview

Amazon S3 (Simple Storage Service) is the foundational object storage service that powers cloud-native applications. It provides virtually unlimited scalability, high durability, and cost-effective storage for any amount of data. Understanding S3 is essential for building modern, cloud-native systems.

This guide explores S3 fundamentals, bucket management, object operations, access control, and practical implementation strategies.

## Problem Statement

Traditional file storage faces challenges:

**Limited Scalability**: File systems hit capacity limits; can't scale horizontally.

**Single Point of Failure**: Data loss risk from hardware failure.

**Expensive**: High upfront costs for dedicated storage arrays.

**Complex Management**: Managing file servers requires significant effort.

S3 solves all of these with virtually unlimited, durable, managed storage.

## Bucket Operations

### Creating a Bucket

```java
@Repository
public class S3BucketService {
    
    @Autowired
    private AmazonS3 amazonS3;
    
    public void createBucket(String bucketName, String region) {
        if (!amazonS3.doesBucketExistV2(bucketName)) {
            CreateBucketRequest request = CreateBucketRequest.builder()
                .bucket(bucketName)
                .createBucketConfiguration(b -> 
                    b.withLocationRegion(region))
                .build();
            
            amazonS3.createBucket(request);
        }
    }
}
```

### Bucket Configuration

```java
public class BucketConfiguration {
    
    // Versioning
    amazonS3.putBucketVersioning(bucketName, VersioningConfiguration.builder()
        .withStatus(VersioningStatus.ENABLED)
        .build());
    
    // Lifecycle
    amazonS3.putBucketLifecycleConfiguration(bucketName, 
        LifecycleConfiguration.builder()
            .withRules(LifecycleRule.builder()
                .withId("archive-old-logs")
                .withFilter(LifecycleRuleFilter.builder()
                    .withPrefix("logs/")
                    .build())
                .withTransitions(LifecycleRuleTransition.builder()
                    .withStorageClass(StorageClass.GLACIER)
                    .withDays(365)
                    .build())
                .withExpirationInDays(2555)
                .build())
            .build())
        .build());
    
    // CORS
    amazonS3.setBucketCors(bucketName, CORSConfiguration.builder()
        .withRules(CORSRule.builder()
            .withAllowedMethods(GET, PUT, POST)
            .withAllowedOrigins("*")
            .withAllowedHeaders("*")
            .withMaxAgeSeconds(3000)
            .build())
        .build());
}
```

## Object Operations

### Upload Objects

```java
@Service
public class S3UploadService {
    
    // Simple upload
    public PutObjectResult upload(String bucket, String key, File file) {
        return amazonS3.putObject(bucket, key, file);
    }
    
    // Upload with metadata
    public PutObjectResult upload(String bucket, String key, File file, 
                           Map<String, String> metadata) {
        ObjectMetadata objectMetadata = new ObjectMetadata();
        objectMetadata.setContentLength(file.length());
        objectMetadata.setContentType("image/jpeg");
        metadata.forEach(objectMetadata::addUserMetadata);
        
        return amazonS3.putObject(bucket, key, file, objectMetadata);
    }
    
    // Multipart upload for large files
    public String multipartUpload(String bucket, String key, File file) {
        InitiateMultipartUploadRequest initRequest = 
            new InitiateMultipartUploadRequest(bucket, key);
        InitiateMultipartUploadResult initResult = 
            amazonS3.initiateMultipartUpload(initRequest);
        
        List<PartETag> partETags = new ArrayList<>();
        long fileSize = file.length();
        long partSize = 5 * 1024 * 1024; // 5MB parts
        
        int partNumber = 1;
        try (InputStream input = new FileInputStream(file)) {
            byte[] buffer = new byte[(int) partSize];
            int read;
            
            while ((read = input.read(buffer)) != -1) {
                UploadPartRequest uploadRequest = UploadPartRequest.builder()
                    .bucket(bucket)
                    .key(key)
                    .uploadId(initResult.getUploadId())
                    .partNumber(partNumber)
                    .inputStream(new ByteArrayInputStream(buffer, 0, read))
                    .partSize(read)
                    .build();
                
                partETags.add(amazonS3.uploadPart(uploadRequest)
                    .getPartETag());
                partNumber++;
            }
        }
        
        CompleteMultipartUploadRequest completeRequest = 
            CompleteMultipartUploadRequest.builder()
                .bucket(bucket)
                .key(key)
                .uploadId(initResult.getUploadId())
                .withPartETags(partETags)
                .build();
        
        amazonS3.completeMultipartUpload(completeRequest);
        return initResult.getUploadId();
    }
}
```

### Download Objects

```java
@Service
public class S3DownloadService {
    
    // Get object
    public S3Object getObject(String bucket, String key) {
        GetObjectRequest request = GetObjectRequest.builder()
            .bucket(bucket)
            .key(key)
            .build();
        
        S3Object s3Object = amazonS3.getObject(request);
        
        return S3Object.builder()
            .key(key)
            .data(s3Object.getObjectContent().readAllBytes())
            .lastModified(s3Object.getObjectMetadata().getLastModified())
            .contentType(s3Object.getObjectMetadata().getContentType())
            .build();
    }
    
    // Get with range (for resumable downloads)
    public InputStream getPartialObject(String bucket, String key, 
                                  long start, long end) {
        GetObjectRequest request = GetObjectRequest.builder()
            .bucket(bucket)
            .key(key)
            .withRange(start, end)
            .build();
        
        return amazonS3.getObject(request).getObjectContent();
    }
}
```

### List Objects

```java
@Service
public class S3ListService {
    
    public List<S3ObjectSummary> listObjects(String bucket, 
                                          String prefix) {
        ListObjectsV2Request request = ListObjectsV2Request.builder()
            .bucket(bucket)
            .prefix(prefix)
            .withMaxKeys(1000)
            .build();
        
        List<S3ObjectSummary> allObjects = new ArrayList<>();
        
        do {
            ListObjectsV2Result result = amazonS3.listObjectsV2(request);
            
            allObjects.addAll(result.getObjectSummaries());
            
            request.setContinuationToken(result.getNextContinuationToken());
        } while (request.getContinuationToken() != null);
        
        return allObjects;
    }
}
```

### Pre-Signed URLs

```java
@Service
public class S3PreSignedUrlService {
    
    // Generate pre-signed URL
    public String generateGetUrl(String bucket, String key, 
                              Duration expiration) {
        GetObjectRequest getRequest = GetObjectRequest.builder()
            .bucket(bucket)
            .key(key)
            .build();
        
        return amazonS3.presignGetObject(getRequest, expiration);
    }
    
    public String generatePutUrl(String bucket, String key, 
                               Duration expiration,
                               String contentType) {
        PutObjectRequest putRequest = PutObjectRequest.builder()
            .bucket(bucket)
            .key(key)
            .contentType(contentType)
            .build();
        
        return amazonS3.presignPutObject(putRequest, expiration);
    }
}
```

## Versioning

### Enable Versioning

```java
@Service
public class S3VersioningService {
    
    public void enableVersioning(String bucket) {
        VersioningConfiguration config = VersioningConfiguration.builder()
            .withStatus(VersioningStatus.ENABLED)
            .build();
        
        amazonS3.setBucketVersioning(bucket, config);
    }
    
    public void listVersions(String bucket, String prefix) {
        ListVersionsRequest request = ListVersionsRequest.builder()
            .bucket(bucket)
            .prefix(prefix)
            .build();
        
        List<S3VersionSummary> versions = 
            amazonS3.listVersions(request).getVersionSummaries();
        
        for (S3VersionSummary version : versions) {
            log.info("Key: {}, Version: {}", 
                version.getKey(), version.getVersionId());
        }
    }
}
```

## Lifecycle Management

### Lifecycle Rules

```java
@Service
public class S3LifecycleService {
    
    public void configureLifecycle(String bucket) {
        LifecycleConfiguration config = LifecycleConfiguration.builder()
            .withRules(
                // Move to Glacier after 90 days
                LifecycleRule.builder()
                    .withId("archive-logs")
                    .withFilter(LifecycleRuleFilter.builder()
                        .withPrefix("logs/")
                        .build())
                    .withTransitions(
                        LifecycleRuleTransition.builder()
                            .withStorageClass(StorageClass.INTELLIGENT_TIERING)
                            .withDays(30)
                            .build(),
                        LifecycleRuleTransition.builder()
                            .withStorageClass(StorageClass.GLACIER)
                            .withDays(90)
                            .build()
                    )
                    .withExpirationInDays(2555)
                    .build(),
                
                // Delete old multi-part uploads
                LifecycleRule.builder()
                    .withId("cleanup-uploads")
                    .withFilter(LifecycleRuleFilter.builder()
                        .build())
                    .withAbortIncompleteMultipartUpload(
                        AbortIncompleteMultipartUpload.builder()
                            .withDaysAfterInitiation(7)
                            .build()
                    )
                    .build()
            )
            .build();
        
        amazonS3.setBucketLifecycleConfiguration(bucket, config);
    }
}
```

### Multi-Part Upload

```java
public class MultiPartUpload {
    
    // For files > 5MB, always use multipart
    public void uploadLargeFile(String bucket, String key, File file) {
        long fileSize = file.length();
        
        if (fileSize > 5 * 1024 * 1024) {
            // Use multipart upload
            multipartUpload(bucket, key, file);
        } else {
            // Simple upload
            amazonS3.putObject(bucket, key, file);
        }
    }
}
```

## Best Practices

1. **Use meaningful prefixes**: Organize data logically.

2. **Enable versioning**: For critical data.

3. **Lifecycle policies**: Auto-archive to Glacier.

4. **Pre-signed URLs**: For secure temporary access.

5. **Multipart uploads**: Always for files > 5MB.

6. **Server-side encryption**: Enable by default.

7. **Cross-region replication**: For disaster recovery.

## Common Mistakes

1. **Making buckets public**: Accidentally exposing data.

2. **Not enabling versioning**: Lost data can't be recovered.

3. **No lifecycle policies**: Accumulating unnecessary costs.

4. **Large files as single upload**: Slow and unreliable.

5. **No access control**: Overly permissive policies.

## Summary

S3 is the foundation of cloud storage. Key practices: use prefixes for organization, enable versioning for critical data, configure lifecycle policies for cost optimization, use multipart for large files, and carefully control access with IAM policies. Combined with CloudFront CDN and cross-region replication, S3 provides the durable, scalable storage foundation for cloud applications.
