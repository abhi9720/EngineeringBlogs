# Tic-Tac-Toe Game - LLD

## Overview

Tic-Tac-Toe is a classic two-player game played on a 3x3 grid. Players take turns marking cells with X or O. The first player to get three of their marks in a horizontal, vertical, or diagonal row wins. Despite its simplicity, designing Tic-Tac-Toe with clean OOP principles demonstrates fundamental concepts like encapsulation, separation of concerns, and state management.

This blog presents a complete low-level design with Java implementation, supporting extensibility for different board sizes and player types.

---

## Problem Statement

Design a Tic-Tac-Toe game that supports:

- Two players playing on a configurable board (default 3x3)
- Human vs Human or Human vs Computer modes
- Move validation (cell must be empty and within bounds)
- Win detection (row, column, diagonal)
- Draw detection (board full with no winner)
- Game state tracking (in progress, won, draw)
- Clear separation of concerns between board, player, game logic

```java
public class Player {

    private final PlayingPiece piece;
    private final PlayerType type;

        this.name = name;
        this.piece = piece;
        this.type = type;
    }

        return name;

    public PlayingPiece getPiece() {
        return piece;

        return type;

// Computer player with simple AI
public class ComputerPlayer extends Player {
    public ComputerPlayer(PlayingPiece piece) {
        super("Computer", piece, PlayerType.COMPUTER);
    }

    public int[] getMove(Board board) {
        // Simple AI: Try to win, then block, then take center/corner
        int size = board.getSize();

        // Try to win
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (board.isCellEmpty(i, j)) {
                    board.placePiece(i, j, getPiece());
                    if (WinChecker.checkWin(board, getPiece())) {
                        board.undoMove(i, j);
                        return new int[]{i, j};
                    }
                    board.undoMove(i, j);
                }
            }
        }

        // Block opponent winning move
        PlayingPiece opponentPiece = getPiece().getPieceType() == PieceType.X
            ? new PlayingPiece(PieceType.O)
            : new PlayingPiece(PieceType.X);

        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (board.isCellEmpty(i, j)) {
                    board.placePiece(i, j, opponentPiece);
                    if (WinChecker.checkWin(board, opponentPiece)) {
                        board.undoMove(i, j);
                        return new int[]{i, j};
                    }
                    board.undoMove(i, j);
                }
            }
        }

        // Take center if available
        int center = size / 2;
        if (board.isCellEmpty(center, center)) {
            return new int[]{center, center};
        }

        // Take a corner
        int[][] corners = {{0, 0}, {0, size - 1}, {size - 1, 0}, {size - 1, size - 1}};
        for (int[] corner : corners) {
            if (board.isCellEmpty(corner[0], corner[1])) {
                return corner;
            }
        }

        // Take any available cell
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (board.isCellEmpty(i, j)) {
                    return new int[]{i, j};
                }
            }
        }

        return new int[]{-1, -1}; // No move available
    }
}
```

### Board

```java
public class Board {
    private final int size;
    private final PlayingPiece[][] grid;

    public Board(int size) {
        this.size = size;
        this.grid = new PlayingPiece[size][size];
    }

    public synchronized boolean placePiece(int row, int col, PlayingPiece piece) {
        if (row < 0 || row >= size || col < 0 || col >= size) {
            System.out.println("Invalid position: (" + row + ", " + col + ")");
            return false;
        }
        if (grid[row][col] != null) {
            System.out.println("Cell (" + row + ", " + col + ") is already occupied");
            return false;
        }
        grid[row][col] = piece;
        return true;
    }

    public void undoMove(int row, int col) {
        grid[row][col] = null;
    }

    public boolean isCellEmpty(int row, int col) {
        return grid[row][col] == null;
    }

    public boolean isFull() {
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (grid[i][j] == null) {
                    return false;
                }
            }
        }
        return true;
    }

    public void display() {
        System.out.println("Current Board:");
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                if (grid[i][j] != null) {
                    System.out.print(" " + grid[i][j].getPieceType().getSymbol() + " ");
                } else {
                    System.out.print("   ");
                }
                if (j < size - 1) {
                    System.out.print("|");
                }
            }
            System.out.println();
            if (i < size - 1) {
                for (int j = 0; j < size; j++) {
                    System.out.print("---");
                    if (j < size - 1) {
                        System.out.print("+");
                    }
                }
                System.out.println();
            }
        }
    }

    public int getSize() { return size; }
    public PlayingPiece[][] getGrid() { return grid; }
}
```

### WinChecker

```java
public class WinChecker {

    public static boolean checkWin(Board board, PlayingPiece piece) {
        return checkRows(board, piece)
            || checkColumns(board, piece)
            || checkDiagonals(board, piece);
    }

    private static boolean checkRows(Board board, PlayingPiece piece) {
        for (int i = 0; i < board.getSize(); i++) {
            boolean win = true;
            for (int j = 0; j < board.getSize(); j++) {
                if (board.getGrid()[i][j] == null
                    || board.getGrid()[i][j].getPieceType() != piece.getPieceType()) {
                    win = false;
                    break;
                }
            }
            if (win) return true;
        }
        return false;
    }

    private static boolean checkColumns(Board board, PlayingPiece piece) {
        for (int j = 0; j < board.getSize(); j++) {
            boolean win = true;
            for (int i = 0; i < board.getSize(); i++) {
                if (board.getGrid()[i][j] == null
                    || board.getGrid()[i][j].getPieceType() != piece.getPieceType()) {
                    win = false;
                    break;
                }
            }
            if (win) return true;
        }
        return false;
    }

    private static boolean checkDiagonals(Board board, PlayingPiece piece) {
        boolean mainDiag = true;
        boolean antiDiag = true;
        int n = board.getSize();

        for (int i = 0; i < n; i++) {
            if (board.getGrid()[i][i] == null
                || board.getGrid()[i][i].getPieceType() != piece.getPieceType()) {
                mainDiag = false;
            }
            if (board.getGrid()[i][n - 1 - i] == null
                || board.getGrid()[i][n - 1 - i].getPieceType() != piece.getPieceType()) {
                antiDiag = false;
            }
        }
        return mainDiag || antiDiag;
    }
}
```

### Game Controller

```java
public class Game {
    private final Board board;
    private final List<Player> players;
    private Player currentPlayer;
    private GameStatus status;
    private Player winner;
    private int moveCount;

    public Game(int size, Player player1, Player player2) {
        this.board = new Board(size);
        this.players = Arrays.asList(player1, player2);
        this.currentPlayer = player1;
        this.status = GameStatus.IN_PROGRESS;
        this.moveCount = 0;
    }

    public GameStatus play(int row, int col) {
        if (status != GameStatus.IN_PROGRESS) {
            System.out.println("Game is already over!");
            return status;
        }

        // Validate move
        if (!board.placePiece(row, col, currentPlayer.getPiece())) {
            return GameStatus.IN_PROGRESS; // Invalid move, try again
        }

        moveCount++;
        board.display();

        // Check for win
        if (WinChecker.checkWin(board, currentPlayer.getPiece())) {
            status = GameStatus.WON;
            winner = currentPlayer;
            System.out.println(currentPlayer.getName() + " wins!");
            return status;
        }

        // Check for draw
        if (board.isFull()) {
            status = GameStatus.DRAW;
            System.out.println("Game is a draw!");
            return status;
        }

        // Switch player
        switchPlayer();
        return status;
    }

    private void switchPlayer() {
        currentPlayer = (currentPlayer == players.get(0)) ? players.get(1) : players.get(0);
        System.out.println(currentPlayer.getName() + "'s turn (" + currentPlayer.getPiece().getPieceType().getSymbol() + ")");
    }

    public void playComputerMove() {
        if (currentPlayer.getType() == PlayerType.COMPUTER) {
            ComputerPlayer computer = (ComputerPlayer) currentPlayer;
            int[] move = computer.getMove(board);
            if (move[0] != -1) {
                play(move[0], move[1]);
            }
        }
    }

    public boolean isGameOver() {
        return status != GameStatus.IN_PROGRESS;
    }

    public Board getBoard() {
        return board;
    }

    public Player getWinner() {
        return winner;
    }

    public Player getCurrentPlayer() {
        return currentPlayer;
    }
}
```

### Demo

```java
public class TicTacToeDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter board size (default 3): ");
        int size = scanner.nextInt();
        scanner.nextLine(); // consume newline

        Player human = new Player("Player 1", new PlayingPiece(PieceType.X), PlayerType.HUMAN);
        Player computer = new ComputerPlayer(new PlayingPiece(PieceType.O));

        Game game = new Game(size, human, computer);

        while (!game.isGameOver()) {
            if (game.getCurrentPlayer().getType() == PlayerType.HUMAN) {
                System.out.print("Enter your move (row col): ");
                int row = scanner.nextInt();
                int col = scanner.nextInt();
                game.play(row, col);
            } else {
                game.playComputerMove();
            }
        }

        if (game.getWinner() != null) {
            System.out.println("Winner: " + game.getWinner().getName());
        } else {
            System.out.println("Game ended in a draw.");
        }

        scanner.close();
    }
}
```

## Best Practices

- Keep the Board class responsible only for grid management, not game logic (Single Responsibility)
- Use enums instead of strings for piece types to prevent invalid states
- Make WinChecker methods static and stateless for reusability
- Implement the undo operation on Board to support AI lookahead without deep copies
- Use immutable PlayingPiece objects to prevent accidental modification
- Separate computer player logic via inheritance for clean extensibility
- Validate all user input before modifying game state

---

## Common Mistakes

- Using strings directly for player symbols instead of enums, risking typos
- Placing win detection logic inside the Board class, violating separation of concerns
- Not handling the undo operation properly in AI implementations
- Allowing moves after the game has ended
- Forgetting to check all winning conditions (rows, columns, both diagonals)
- Using a flat array instead of a 2D grid, reducing code readability
- Not validating that the correct player is making a move

---

## Summary

The Tic-Tac-Toe design demonstrates clean separation of concerns through distinct classes for Board, Player, Game, and WinChecker. The use of enums ensures type safety for piece types and game states. The inheritance-based approach for ComputerPlayer makes the system extensible for different AI strategies. By following OOP principles and applying the Single Responsibility Principle, the resulting design is easy to understand, test, and extend to support larger boards or alternative game rules.
