# AI/ML Learning Roadmap

## Overview

This roadmap guides you through mastering AI and Machine Learning, starting from basics to advanced production systems. Each section builds on the previous, ensuring solid foundations before moving forward.

**Total sections:** 8  
**Estimated reading time:** 40+ hours  
**Prerequisites:** Basic programming knowledge

## Section 1: Foundations (01-foundations/)

**Goal:** Understand what AI/ML is and learn core concepts

**Topics:**
- What is AI and ML?
- Types of Machine Learning
- Supervised vs Unsupervised Learning
- ML Lifecycle
- Bias-Variance Tradeoff
- Model Evaluation Metrics

**Time:** 4-5 hours  
**Difficulty:** Beginner

**Start here if:** You're new to ML or need a refresher on fundamentals.

---

## Section 2: Deep Learning Core (02-deep-learning-core/)

**Goal:** Master neural networks from basics to transformers

**Topics:**
- Neural Networks Basics
- Backpropagation Explained
- CNNs for Computer Vision
- RNNs for Sequence Data
- Attention Mechanism
- Transformers Architecture
- Embeddings in Deep Learning

**Time:** 8-10 hours  
**Difficulty:** Intermediate

**Start here if:** You understand ML basics and want to learn deep learning.

---

## Section 3: LLM Foundations (03-llm-foundations/)

**Goal:** Understand how LLMs work internally

**Topics:**
- Introduction to LLMs
- Tokenization and Context Windows
- Embeddings Explained

- Semantic Search Basics
- Vector Databases Intro

**Time:** 5-6 hours  
**Difficulty:** Intermediate

**Start here if:** You know deep learning basics and want to understand LLMs.

## Section 5: RAG and Retrieval Systems (05-rag-and-retrieval-systems/)

- Memory in Agents

**Frameworks:**
- LangChain Introduction
- LangGraph State Machines
- AutoGen Multi-Agent
- CrewAI Role-Based Agents
- Semantic Kernel Overview

**Multi-Agent Systems:**
- Multi-Agent Overview
- Coordination Strategies
- Role-Based Agents
- Swarm vs Decentralized Agents

**Tools & Memory:**
- Tool Use in Agents
- Memory Types (Short/Long Term)

- Evaluating Agent Performance

## Learning Tips

1. **Build projects**: Don't just read - implement what you learn
2. **Read papers**: Start with seminal papers, then recent research
3. **Join communities**: r/MachineLearning, Discord servers, Twitter
4. **Practice interviews**: Use this content for interview prep
5. **Teach others**: Explaining concepts helps cement your understanding
